# Naming Rules

## Symbols
Use clear names like:
- `MCU_STM32G431CBUx`
- `CONN_USB_C_16P`

## Footprints
Use clear names like:
- `QFN-48_7x7mm_P0.5mm`
- `USB_C_Receptacle_HRO_TYPE-C-31-M-12`

## Rules
- No spaces
- No vague names like `new_part` or `test1`
- Keep symbol and footprint names easy to search
