# Review Checklist

## Symbol
- [ ] Pin numbers match datasheet
- [ ] Pin names match datasheet
- [ ] Symbol name is clear

## Footprint
- [ ] Dimensions match datasheet
- [ ] Pad sizes are correct
- [ ] Pin 1 is clear
- [ ] Silkscreen is acceptable

## Final
- [ ] Datasheet link recorded
- [ ] Reviewed by one other person
- [ ] Moved from draft to approved
